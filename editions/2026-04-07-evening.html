The move comes as AI model capabilities increasingly overlap, making the economics of distillation — building a capable model by learning from a more expensive one — particularly attractive to well-resourced adversaries.

Cursor warp decode achieves 1.84x faster MoE inference by eliminating expert-batching overhead

Cursor rewrote its mixture-of-experts token generation pipeline with "warp decode" — assigning each GPU warp directly to one output value instead of grouping work by experts. The result: 1.84x higher throughput on Blackwell GPUs, memory bandwidth approaching ~3.95 TB/s, and outputs 1.4x closer to FP32 reference precision than standard MoE decode.

Standard MoE decoding wastes cycles on scatter/combine stages and padding that exist purely to coordinate around expert groupings. Warp decode eliminates these stages entirely by mapping one warp to one output neuron, fusing the operation into two GPU kernels that stream weights, accumulate in registers, and write a single output. This reduces intermediate buffer use and GPU synchronization overhead simultaneously. The breakthrough also demonstrates how inference optimization at the infrastructure layer — not model architecture — is increasingly where competitive advantage is being built. AlphaSignal notes this is a top-trending technical finding with 2,528 stars on GitHub.

MemPalace: Milla Jovovich's open-source AI memory layer scores 100% on LongMemEval benchmark

Actress and entrepreneur Milla Jovovich released MemPalace, an open-source memory layer for LLM workflows that organizes full conversation history into a hierarchical "palace" structure — wings (projects), rooms (topics), halls (types) — achieving a perfect 500/500 on LongMemEval and 96.6% without API calls using local retrieval. It has 9,528 GitHub stars.

MemPalace solves the "disappearing context" problem by storing full conversations rather than summaries, then using structural narrowing before semantic search to retrieve the right context fast. Its AAAK compression achieves ~30x smaller context that remains readable by any LLM. The local stack (SQLite + ChromaDB) requires no API key, and a knowledge graph layer tracks facts and flags contradictions. The development underscores the point made in the Claude Code source code analysis: the wrapper and infrastructure layer is where the next wave of AI differentiation is being built, and that's now happening outside traditional AI labs.
