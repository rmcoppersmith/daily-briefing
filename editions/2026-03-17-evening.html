OpenAI Introduces Subagents in Codex for Parallel Processing

OpenAI's Codex now lets users spawn specialized subagents that work on different parts of a task simultaneously. Each subagent operates with its own instructions, model settings, and tool context before merging results into a unified output.

The subagent feature is perfect for complex workflows like PR reviews, codebase exploration, and multi-step debugging where parallel processing improves efficiency. Instead of one agent grinding through a task sequentially, the system can deploy a repo scanner, patch handler, and review agent all at once. This architecture mirrors enterprise team structures and enables more sophisticated code generation at scale.

Apple Launches AirPods Max 2 with H2 Chip and Live Translation

Apple unveiled AirPods Max 2, its first premium headphone update since 2020, featuring the H2 chip, improved active noise cancellation, and built-in live translation powered by Apple Intelligence. Pre-orders open March 25 with availability in early April.

The new model includes voice isolation for calls, Spatial Audio improvements, and a remote camera feature. Apple's focus on AI-powered translation and intelligence features positions the device as a personal AI assistant in hardware form. The update demonstrates Apple's commitment to integrating on-device AI into premium products while maintaining focus on privacy and user experience.
