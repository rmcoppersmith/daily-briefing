Axios Supply Chain Attack Compromises npm Package with 83M Weekly Downloads

Attackers hijacked a maintainer's GitHub account for the Axios JavaScript library, slipping malware into two new releases that deploy a remote access trojan across macOS, Windows, and Linux. At least 135 compromised machines have been identified so far.

The malicious code was injected into versions 1.14.1 and 0.30.4, quietly deploying a RAT before wiping its own tracks to evade detection. Security experts urge an immediate downgrade and full credential rotation on any affected machine. With over 83 million weekly downloads, Axios is one of the most widely used JavaScript packages, making the blast radius of this supply chain attack potentially enormous.
