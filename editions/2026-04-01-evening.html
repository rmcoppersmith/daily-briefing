ARC-AGI-3 Drops Every Major AI Model to Below 1% — Humans Score 100%

The ARC Prize Foundation released ARC-AGI-3, the first interactive reasoning benchmark for AI agents. It drops models into unfamiliar game worlds with no instructions, rules, or goals — and every frontier model failed catastrophically. Gemini 3.1 Pro led at just 0.37%, GPT-5.4 scored 0.26%, Claude Opus 4.6 hit 0.25%, and Grok-4.20 scored 0%. Untrained humans solve 100% of environments.

The benchmark was built by François Chollet and Mike Knoop's foundation, which set up an in-house game studio and created 135 original interactive environments from scratch. The design is intentional: an AI agent must explore, figure out what it's supposed to do, form a plan, and execute it — testing in-context learning and adaptation rather than memorization. Today's frontier models are, in Chollet's framing, "incredible memorization machines" that can ace trained-on exams but fail when faced with genuinely novel environments. The competition runs through December 2026 with an $850,000 prize pool ($700K grand prize for the first agent to score 100%). The benchmark is open to the public — you can play it yourself at arcprize.org/tasks.

Korean Developer Rewrites Leaked Claude Code in Python Overnight — "claw-code" Hits 50K Stars Before Anthropic DMCA

After Anthropic began issuing DMCA takedowns across GitHub mirrors of its leaked source code, Korean developer Sigrid Jin woke at 4 AM and rewrote the entire Claude Code codebase from scratch in Python. The incident underscores a quirk of copyright law: a clean-room rewrite that doesn't copy the original code directly is protected, regardless of what inspired it.
