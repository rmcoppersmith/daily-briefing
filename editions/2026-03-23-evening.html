Supermemory Hits 99% Accuracy With New Agent Memory System — No Vector Database Needed

AI startup Supermemory unveiled ASMR (Agentic Search and Memory Retrieval), a system that replaces vector search with parallel teams of specialized agents that actively read, analyze, and extract facts from conversation histories. Their LongMemEval benchmark score jumped from 85% to 98.6%.

MCP Backlash Grows — Perplexity Ditches the Protocol as Devs Debate Its Value

A growing number of developers are questioning whether Anthropic's Model Context Protocol adds real value or just unnecessary overhead. The debate hit a tipping point when Perplexity co-founder Denis Yarats revealed the company is abandoning MCP in favor of direct APIs and CLIs.

Critics argue that CLI tools like curl and git already exist in model training data, so agents can use them without schema overhead. But the backlash has a blind spot: almost no one separates MCP's two modes. Local stdio keeps the server on your machine, while remote HTTP puts it on a centralized server for team-wide access. Centralized MCP gives teams one place to manage auth, track tools, and maintain consistency across agents — problems CLIs were never built to solve. The smartest teams aren't picking sides; they use CLIs when a model already understands the tool and pivot to MCP for enterprise auth, observability, and governance.

AI Cattle Startup Halter Gets Founders Fund Backing for Its 'Cowgorithm'

Halter, an AI startup backed by Peter Thiel's Founders Fund, makes solar-powered GPS collars that herd cattle using sound and vibration cues — eliminating fences, farm dogs, and manual herding. The collar also monitors digestion, fertility cycles, and biometrics around the clock.

While the national AI conversation focuses on office productivity, some of the most tangible gains are showing up in pastures.
